# Contributing to coreX AI MindLayer

We welcome contributions! Please ensure your code follows the "Compass, not Chronicle" principle.

## Development Setup
1. Clone the repository
2. `cd vscode-extension`
3. `npm install`
4. Use `F5` in VS Code to start the extension development host.

## Rules
- Keep the core lightweight.
- No auto-writing to project files without user consent.
- Always include JSON Schema validation for new file types.
